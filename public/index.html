short

Introduction

A short url service.

Getting started

Endpoints

GET /new

Create a new short url.

$ curl https://t.zce.me/new

Parameters

  • url: target url, alias: link
  • id: short id, default: auto

Response

{
  "id": "<shortid>",
  "link": "http://t.zce.me/<shortid>"
}
